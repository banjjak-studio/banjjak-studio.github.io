모아 개인정보 처리방침

시행일: 2026년 5월 18일

모아(MOA)는 가계부, 구독, 목표, 그룹 기반 공동 관리를 돕는 모바일 앱입니다. 본 개인정보 처리방침은 모아 앱과 모아 API 서비스에서 처리하는 개인정보와 데이터의 이용 방식을 설명합니다.

1. 수집하는 개인정보

모아는 서비스 제공을 위해 아래 정보를 수집하거나 저장할 수 있습니다.

  • 계정 정보: 이메일 주소, 표시 이름 또는 닉네임
  • 인증 정보: 이메일 인증 코드 요청 및 검증 기록, 로그인 세션 토큰
  • 그룹 정보: 그룹명, 멤버, 역할, 초대 이메일, 초대 상태
  • 가계부 정보: 사용자가 입력한 수입, 지출, 카테고리, 메모, 날짜, 결제자, 반복 규칙, 자주 쓰는 입력
  • 구독 정보: 구독명, 금액, 결제 주기, 결제일, 담당자, 관리 작업
  • 목표 및 금고 정보: 목표명, 목표 금액, 적립/이체 기록
  • 가져오기 정보: 사용자가 직접 선택한 CSV 또는 백업 JSON 파일에서 확인 후 등록한 거래, 카테고리, 목표, 설정 데이터
  • 설정 정보: 앱 설정, 그룹 설정, 카테고리, 알림 선호 설정
  • 운영 정보: API 요청 시각, 오류 로그, 보안 및 부정 이용 방지에 필요한 기록

2. 개인정보 이용 목적

  • 이메일 인증을 통한 로그인과 계정 식별
  • 가계부, 구독, 고정 지출, 목표, 금고 기능 제공
  • 그룹 멤버 초대, 공동 기록 조회와 관리
  • 사용자 설정 저장과 여러 기기 간 데이터 동기화
  • 서비스 장애 분석, 보안, 부정 이용 방지
  • 사용자 문의 대응과 계정 삭제 요청 처리

3. 앱 권한과 기기 정보

현재 모아 앱은 Android에서 별도의 민감 권한을 요청하지 않습니다. 앱은 로그인 세션 유지를 위해 기기 보안 저장소에 인증 토큰을 저장합니다. CSV 또는 백업 JSON 가져오기는 사용자가 파일을 직접 선택한 경우에만 파일 내용을 읽고, 사용자가 등록 또는 복원을 확정한 데이터만 서버로 전송합니다. 향후 푸시 알림 기능을 제공하는 경우 알림 권한을 요청할 수 있으며, 이때 푸시 토큰과 기기명이 저장될 수 있습니다.

4. 제3자 제공 및 위탁

모아는 사용자의 개인정보를 판매하지 않습니다. 서비스 운영을 위해 아래 외부 서비스를 이용할 수 있습니다.

  • 이메일 발송 서비스: 로그인 인증 코드와 그룹 초대 메일 발송
  • 호스팅 및 인프라 서비스: API 서버, 데이터베이스, 로그 저장
  • Google Play: 앱 배포, 설치 및 오류 통계 제공

5. 보관 및 삭제

계정과 사용자가 입력한 데이터는 서비스 제공 기간 동안 보관됩니다. 사용자가 계정 삭제를 요청하면 로그인 세션은 해제되고 계정 식별 정보는 삭제 또는 익명화됩니다. 단, 공동 그룹에서 다른 멤버와 함께 작성한 기록은 다른 멤버의 이용 내역 보존을 위해 일부 남을 수 있습니다.

이메일로 접수된 삭제 요청 메일과 본인 확인 기록은 처리 완료 후 최대 30일 동안 보관될 수 있습니다. 보안, 오남용 방지, 분쟁 대응에 필요한 비식별 운영 로그는 최대 90일 동안 보관될 수 있으며, 관계 법령상 더 긴 보관 의무가 있는 경우 해당 법정 기간 동안만 보관됩니다.

계정 및 데이터 삭제 방법은 모아 계정 및 데이터 삭제 안내에서 확인할 수 있습니다.

6. 사용자의 권리

사용자는 앱 내 설정에서 계정 삭제를 요청할 수 있으며, 문의 이메일을 통해 개인정보 열람, 정정, 삭제, 처리 정지를 요청할 수 있습니다. 이메일 요청의 본인 확인은 모아 로그인 이메일 주소를 실제로 사용할 수 있는지 확인하는 방식으로 진행하며, 같은 이메일 주소로 보낸 삭제 확인 메일에 사용자가 회신한 뒤 처리합니다.

7. 아동 개인정보

모아는 만 14세 이상 사용자를 대상으로 합니다. 만 14세 미만 아동의 개인정보를 고의로 수집하지 않으며, 관련 사실을 확인한 경우 필요한 조치를 진행합니다.

8. 보안

모아는 HTTPS 통신, 인증 토큰 관리, 접근 제어, 데이터베이스 백업 등 합리적인 보호 조치를 적용합니다. 다만 인터넷 기반 서비스의 특성상 모든 위험을 완전히 제거할 수는 없습니다.

9. 변경 사항

서비스 기능, 수집 항목, 외부 서비스 이용 방식이 변경되는 경우 본 처리방침을 업데이트합니다. 중요한 변경이 있는 경우 앱 또는 웹페이지를 통해 안내합니다.
