모아 계정 및 데이터 삭제 안내

시행일: 2026년 5월 18일

모아(MOA) 사용자는 앱 안에서 계정 삭제를 요청하거나, 이메일로 계정 및 데이터 삭제를 요청할 수 있습니다. 모아는 banjjak studio가 운영합니다.

1. 앱에서 삭제 요청

  1. 모아 앱을 엽니다.
  2. 더보기에서 모아 설정으로 이동합니다.
  3. 데이터 또는 계정 영역에서 계정 삭제 요청을 선택합니다.
  4. 안내 내용을 확인한 뒤 삭제를 확정합니다.

2. 이메일로 삭제 요청

앱에 로그인할 수 없는 경우 아래 이메일로 삭제를 요청할 수 있습니다. 본인 확인은 모아 로그인 이메일 주소를 실제로 사용할 수 있는지 확인하는 방식으로 진행합니다.

  • 문의 이메일: banjjak.support@gmail.com
  • 메일 제목: 모아 계정 삭제 요청
  • 발신 주소: 모아 로그인 이메일과 같은 주소 사용
  • 메일 내용: 모아 로그인 이메일 주소
  • 삭제 요청 사유: 선택 사항

요청이 접수되면 같은 이메일 주소로 삭제 확인 메일을 보냅니다. 사용자가 확인 회신을 보내면 계정 삭제를 처리합니다. 비밀번호, 신분증, 금융 정보, 인증 코드는 이메일로 요청하지 않습니다.

3. 삭제되는 데이터

  • 로그인 이메일과 계정 식별 정보
  • 활성 로그인 세션과 인증 토큰
  • 개인 앱 설정과 알림 설정
  • 기기 푸시 토큰과 알림 수신 설정
  • 해당 계정이 단독으로 보유한 개인 데이터

4. 일부 보관될 수 있는 데이터

공동 그룹에서 다른 멤버와 함께 작성한 거래, 구독, 목표, 그룹 기록은 다른 멤버의 이용 내역 보존을 위해 일부 남을 수 있습니다. 이 경우 삭제 요청자의 로그인 이메일과 이름은 삭제 또는 익명화됩니다.

이메일로 접수된 삭제 요청 메일과 본인 확인 기록은 처리 완료 후 최대 30일 동안 보관될 수 있습니다. 보안, 오남용 방지, 분쟁 대응에 필요한 비식별 운영 로그는 최대 90일 동안 보관될 수 있으며, 관계 법령상 더 긴 보관 의무가 있는 경우 해당 법정 기간 동안만 보관됩니다.

5. 처리 기간

앱 내 삭제는 즉시 반영되는 것을 원칙으로 합니다. 이메일 요청은 본인 확인 회신을 받은 뒤 최대 30일 이내에 처리합니다.

개인정보 처리 기준은 모아 개인정보 처리방침에서 확인할 수 있습니다.
